* DSC Token is an Ethereum based ERC20 token.
* On-platform, users and DiscoverBlockchain partners will transact using the DiscoverBlockchain Coin (DSC) which is a utility token built on the Ethereum Blockchain.
* The creation of DSC is accompanied by Ethereum built Smart Contracts that autonomously execute data and value exchange.
* For DiscoverBlockchain, DSC and Smart Contracts replace fiat currency and privately administered code respectively.
* DSC token is not a security token because token owners do not have voting rights as shareholders, share revenue or receive any dividends.
* There is no expectation of profit from the DSC token. The DSC Token passes the Howey test – check our legal section on the whitepaper to learn more about why we are a utility token.
